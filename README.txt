this file is for the body section of the FEC. it does not include the hero-section or the reviews section.


to seed the database run "npm run seed" in the base terminal

to run the server run "npm start" in the base terminal

to build the bundle, cd into the client folder and run "npm run build"

you may need to run "npm install" in both of the previously stated terminals.

you may need to change the port number in server/index.js so it dosent interfere with your other ports.
